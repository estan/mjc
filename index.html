Welcome!

This is a compiler for a slightly modified version of the MiniJava language described in Appel's Modern Compiler Implementation in Java. It was created as part of the course DD2488 Compiler Construction at KTH.

For compiler sources and more information, see the GitHub repository.

Generated JavaDoc documentation for the compiler sources is available here.
